# Support

Board Mute support는 첫 공개 릴리스에서 GitHub Issues로 받습니다.

## 지원 범위

이슈로 받을 수 있는 내용:

- 지원 사이트의 게시판 목록 row가 숨겨지지 않음
- 작성자 `차단` 버튼이 표시되지 않음
- 빠른 차단 후 row 숨김, 되돌리기, 잠시 보기가 예상대로 동작하지 않음
- popup에서 규칙 추가, 삭제, 검색, 필터가 예상대로 동작하지 않음
- README, PRIVACY, CHANGELOG와 실제 동작이 맞지 않음

현재 지원하지 않는 범위:

- 댓글 작성자 차단
- 모바일 웹
- 글 본문 작성자 차단
- 모든 한국 커뮤니티 자동 지원
- 전역 작성자, 아이디, IP 차단
- 지원 row 구조가 없는 페이지의 별도 고정 관리 버튼

## 이슈 작성 전 확인

1. Chrome `chrome://extensions`에서 Board Mute를 새로고침합니다.
2. 문제가 난 게시판 페이지를 새로고침합니다.
3. popup에서 해당 사이트의 `차단` 토글이 켜져 있는지 확인합니다.
4. 같은 문제가 계속 나면 이슈를 작성합니다.

## 이슈에 적어 주세요

- 문제가 난 사이트
- 문제가 난 페이지 URL
- Board Mute 버전
- Chrome 버전
- 기대한 동작
- 실제 동작
- 재현 단계
- 가능하면 스크린샷

개인정보, 로그인 정보, 비공개 게시글 내용, 원치 않는 작성자 이름은 스크린샷에서 가리거나 지워 주세요.

## DOM 변경 제보

지원 사이트가 화면 구조를 바꾸면 Board Mute가 row, 제목, 작성자 값을 찾지 못할 수 있습니다.

DOM 변경으로 보이는 문제를 제보할 때는 아래 정보를 함께 보내 주세요.

- 이전에는 동작했는지
- 어느 사이트와 게시판에서 바뀌었는지
- 작성자 `차단` 버튼이 사라졌는지
- 제목 키워드 숨김도 같이 실패하는지
- 숨겨야 할 row와 숨기면 안 되는 공지, 광고, 안내 row가 함께 보이는 스크린샷
