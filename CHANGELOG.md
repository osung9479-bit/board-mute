# Changelog

Board Mute의 공개 릴리스 변경 사항을 기록합니다.

## Unreleased

## 0.5.1 - 2026-05-18

### Added

- 지원 사이트 탭에서 popup을 열면 해당 사이트 규칙 화면을 자동으로 먼저 보여주도록 했습니다.
- popup에 `선택 사이트`와 `공통` 규칙 탭을 추가해 12개 지원 사이트 기준 규칙 관리 구조를 더 쉽게 나눴습니다.
- popup 상단에 `12개 지원` 배지를 추가하고, 규칙 탭별 개수 배지를 표시하도록 했습니다.
- 12개 지원 사이트 검증을 한 번에 실행하는 `scripts/check-supported-sites.js`를 추가했습니다.

### Changed

- popup 규칙 탭에 tablist/tab/tabpanel 접근성 구조와 키보드 전환을 추가했습니다.
- 페이지 안 숨김 상태바의 ARIA 상태 요약과 `잠시 보기` 토글 안내를 보강했습니다.
- 12개 사이트 check suite 출력이 기본적으로 compact summary를 보여주고, 필요할 때 `--verbose`로 전체 로그를 볼 수 있게 했습니다.
- README와 SUPPORT의 지원 범위 문구를 채용 목록, 회사/공고 숨김값, 사이트별 규칙 범위가 더 잘 드러나도록 정리했습니다.

### Fixed

- `선택 숨김` 후 toast의 `되돌리기` 버튼이 실제로 임시 숨김을 복구하도록 수정했습니다.
- Windows에서 임시 Chrome profile 삭제가 잠깐 실패해도 check suite가 불안정해지지 않도록 cleanup retry를 보강했습니다.

## 0.5.0 - 2026-05-18

### Added

- 잡코리아 PC 채용 목록 row 숨김 지원을 추가했습니다.
- 잡코리아 회사 ID `company:<token>`, 회사명 fallback, 특정 공고 `gno:<gno>` 숨김값을 빠른 메뉴에서 추가할 수 있게 했습니다.
- 잡코리아에서는 popup과 페이지 UI가 사람인과 같은 `회사/공고 숨김`, `공고/조건 키워드`, `최근 숨김` 문구를 사용하도록 했습니다.
- 잡코리아 채용 목록 대표 URL을 검증하는 `scripts/check-jobkorea.js`를 추가했습니다.
- 인벤 PC 게시판 목록 row 숨김 지원을 추가했습니다.
- 인벤 글쓴이 닉네임 fallback과 특정 글 `post:<boardId>/<postId>` 숨김값을 빠른 메뉴에서 추가할 수 있게 했습니다.
- 인벤 대표 게시판 URL을 검증하는 `scripts/check-inven.js`를 추가했습니다.
- 네이트판 PC 톡톡 목록 row 숨김 지원을 추가했습니다.
- 네이트판 작성자 검색값 fallback과 특정 글 `post:<postId>` 숨김값을 빠른 메뉴에서 추가할 수 있게 했습니다.
- 네이트판 대표 톡톡 목록 URL을 검증하는 `scripts/check-natepann.js`를 추가했습니다.

## 0.4.0 - 2026-05-18

### Added

- 사람인 PC 검색 결과와 직무별 목록 row 숨김 지원을 추가했습니다.
- 사람인 회사 ID `csn:<token>`, 회사명 fallback, 특정 공고 `rec:<rec_idx>` 숨김값을 빠른 메뉴에서 추가할 수 있게 했습니다.
- 사람인에서는 popup과 페이지 UI가 `회사/공고 숨김`, `공고/조건 키워드`, `최근 숨김` 문구를 사용하도록 했습니다.
- 사람인 검색 결과와 직무별 목록 대표 URL을 검증하는 `scripts/check-saramin.js`를 추가했습니다.

## 0.3.0 - 2026-05-17

### Added

- 퀘이사존 PC BBS 게시판 목록 row 숨김 지원을 추가했습니다.
- 퀘이사존 `data-id` 기반 작성자 빠른 차단과 popup 사이트별 규칙 관리를 추가했습니다.
- 퀘이사존 자동 검증과 popup 사이트 전환 검증을 추가했습니다.
- SLR클럽 PC `zboard.php` 게시판 목록 row 숨김 지원을 추가했습니다.
- SLR클럽 `data-xuid` 기반 작성자 빠른 차단과 popup 사이트별 규칙 관리를 추가했습니다.
- SLR클럽 자동 검증과 popup 사이트 전환 검증을 추가했습니다.
- 게시판 페이지에서 요소를 이번 페이지에서만 직접 숨기는 `선택 숨김` 기능을 추가했습니다.

### Changed

- popup의 작성자 규칙 문구를 사이트별 적용 범위가 드러나도록 정리했습니다.
- popup 상단을 선택 사이트, 켜짐/꺼짐 상태, 규칙 개수 중심으로 재정리하고 규칙 섹션의 범위 표시와 삭제 버튼 문구를 개선했습니다.
- 최근 차단 패널의 `해제` 버튼이 클릭 중 작아지거나 처리 상태가 애매해 보이지 않도록 안정화했습니다.
- 페이지 안 숨김 관리 UI를 `차단중` 상태, 숨김 사유 요약, compact action 구조로 정리했습니다.
- 작성자 `차단` 버튼, 작성자 메뉴, toast, 최근 차단 패널의 시각 밀도와 viewport 안정성을 개선했습니다.

## 0.1.0 - 2026-05-16

### Added

- 오늘의유머 PC 홈, 게시판 목록, 글 보기 본문과 하단 목록 row 숨김 지원을 추가했습니다.
- 오늘의유머 `mn` 회원번호 기반 작성자 빠른 차단과 popup 사이트별 규칙 관리를 추가했습니다.
- 오늘의유머 자동 검증과 popup 사이트 전환 검증을 추가했습니다.

### Changed

- 글 보기 본문처럼 큰 row를 차단 해제할 때 화면 기준점이 크게 흔들리지 않도록 복구 흐름을 안정화했습니다.

## 0.0.3 - 2026-05-16

### Fixed

- 클리앙 PC 게시판 목록에서 작성자 빠른 차단 버튼이 닉네임 영역을 흔들지 않도록 row-level action으로 분리했습니다.
- 클리앙 자동 검증이 빠른 차단 메뉴를 열 때 작성자 셀 위치와 크기가 유지되는지 확인하도록 보강했습니다.

## 0.0.2 - 2026-05-16

### Fixed

- 클리앙 PC 게시판 목록에서 차단 저장은 되지만 실제 row가 화면에서 사라지지 않던 문제를 수정했습니다.
- 클리앙 자동 검증이 `hidden` 속성뿐 아니라 실제 렌더링 숨김 상태까지 확인하도록 보강했습니다.

## 0.0.1 - 2026-05-15

### Added

- 디시인사이드 게시판 목록과 글 보기 하단 리스트 row 숨김 지원
- 에펨코리아 일반 게시판 목록과 글 보기 하단 일반 목록 row 숨김 지원
- 루리웹 PC BBS 게시판 목록 row 숨김 지원
- 엠팍 PC 목록 row 숨김 지원
- 클리앙 PC 게시판 목록 row 숨김 지원
- 사이트별 제목 키워드 규칙
- 지원 사이트 전체에 적용되는 전역 제목 키워드 규칙
- 사이트별 작성자, 아이디, IP, 회원 식별값 규칙
- 지원 row의 작성자 영역에서 빠른 차단 메뉴 제공
- 방금 추가한 빠른 차단 규칙 되돌리기
- 숨겨진 row의 잠시 보기와 다시 숨김 컨트롤
- popup의 사이트 선택, 규칙 추가/삭제, 작성자 목록 검색/필터 관리
- 로컬 저장소 `chrome.storage.local` 기반 규칙 저장

### Supported Sites

- 디시인사이드: 게시판 목록, 글 보기 하단 리스트
- 에펨코리아: 일반 게시판 목록, 글 보기 하단 일반 목록
- 루리웹: PC BBS 게시판 목록
- 엠팍: PC 목록
- 클리앙: PC 게시판 목록

### Known Limitations

- 모든 한국 커뮤니티를 자동 지원하지 않습니다.
- 댓글 작성자 차단은 지원하지 않습니다.
- 모바일 웹은 지원하지 않습니다.
- 글 본문 작성자 차단은 지원하지 않습니다.
- 전역 작성자, 아이디, IP 차단은 지원하지 않습니다.
- 현재 탭 URL 기반 popup 자동 사이트 선택은 지원하지 않습니다.
- 지원 row 구조가 없는 페이지에는 별도 고정 관리 버튼을 제공하지 않습니다.
- 사이트 DOM이 바뀌면 adapter와 검증 기준을 다시 확인해야 합니다.
